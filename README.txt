
FuturoVet - Site demo (estático)
--------------------------------
Arquivos no ZIP:
- index.html
- css/style.css
- js/script.js
- README.txt (este arquivo)

Como usar:
1. Extraia o arquivo ZIP para uma pasta local.
2. Abra o arquivo index.html em um navegador moderno (Chrome, Edge, Firefox).
3. Todas as funções são demos e funcionam localmente sem servidor.
4. Para integrar email/agendamentos de verdade, conecte os formulários a um back-end ou serviço de formulário (ex: Formspree, Netlify Forms).
